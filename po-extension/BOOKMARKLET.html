Option A: Chrome Extension (BEST — works automatically)

  1. In Chrome, go to chrome://extensions
  2. Enable Developer mode (top right toggle)
  3. Click Load unpacked
  4. Select the po-extension folder
  5. Open Pocket Option — data captures automatically

How it works

Pocket Option sends live price ticks through a WebSocket connection. This tool intercepts those ticks at the source — the same data that draws the candlestick chart — and forwards them to the RK DXB Trader server. The signal engine then uses real prices (instead of SVG-approximated values) to build better candles and calculate more accurate signals.
